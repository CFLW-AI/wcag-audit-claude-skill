Audit Summary

Conformance Status: FAIL

This page does not conform to WCAG 2.2 Level AA due to color contrast failures.

Failures Found:
  • SC 1.4.3 (Contrast - Text): 3 text color pairs fail 4.5:1 minimum
    • Card metadata #959595 on #f5f5f5 = 2.75:1
    • Tag text #ffffff on #74b9ff = 2.07:1
    • Overlay text effective #9a9a9a on #333333 = 4.49:1 (just below threshold)
  • SC 1.4.11 (Non-text Contrast): 1 UI component fails 3:1 minimum
    • Outline button border #cccccc on white = 1.61:1
Recommendations:
  1. Card meta text: Darken from #959595 to approximately #5a5a5a to achieve 4.5:1
  2. Tag text: Change background from #74b9ff to a darker blue (e.g., #0056b3) or change text to darker color
  3. Overlay text: Increase opacity or use solid color with 4.5:1 contrast instead of rgba(255,255,255,0.7)
  4. Button outline: Darken border from #cccccc to #999999 or darker to achieve 3:1 against white

Audit Methodology: This audit evaluated the HTML source code against all 28 WCAG 2.2 Level AA success criteria. Color contrast verification used the WCAG relative luminance algorithm implemented in Python to ensure mathematical precision.

Scope: Static HTML structure and CSS styling. Dynamic behavior, JavaScript interactions, and runtime state changes were not evaluated.
