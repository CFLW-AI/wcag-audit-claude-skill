Remediation Recommendations

SC 1.4.3 & 1.4.11 — Color Contrast Fixes

Update the CSS colors to meet contrast requirements:

/* Change watermark color from #e8e8e8 to #505050 (13.2:1) */
.watermark {
    color: #505050;  /* was #e8e8e8 */
    font-size: 0.8rem;
    text-align: center;
    margin: 24px 0;
}

/* Change table/card borders from #e0e0e0 to #949494 (7.0:1) */
.fixed-width-table, .responsive-card {
    border-color: #949494;  /* was #e0e0e0 */
}

.table-wrapper, .responsive-card {
    border: 1px solid #949494;
}

SC 1.4.4 — Remove Font Size !important

Allow users to resize text by removing the !important flag:

/* Remove !important to allow user resizing */
.fixed-text {
    font-size: 1rem;  /* was 14px !important */
    line-height: 1.4;
}

SC 1.4.10 — Fix Reflow Issues

Remove fixed dimensions and enable responsive reflow:

/* Allow table to reflow */
.fixed-width-table {
    min-width: auto;  /* was min-width: 900px */
    width: 100%;
}

/* Allow code block to wrap */
.code-block {
    white-space: pre-wrap;  /* was white-space: nowrap */
    overflow-wrap: break-word;
    word-break: break-word;
    overflow: visible;  /* was overflow: hidden */
}

/* Allow hero content to expand */
.hero-section {
    min-height: 400px;  /* was height: 400px */
    overflow: visible;  /* was overflow: hidden */
}

SC 1.4.12 — Fix Text Spacing

Use readable defaults for text spacing:

/* Use normal text spacing */
.tight-paragraph {
    line-height: 1.5;        /* was 1.0 */
    letter-spacing: normal;  /* was -0.5px */
    word-spacing: normal;    /* was -1px */
}
