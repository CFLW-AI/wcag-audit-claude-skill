Understanding Web Standards

Web standards ensure that content is accessible to all users regardless of ability or device.

Historical Context

The web standards movement began in the late 1990s with the founding of the Web Standards Project.

As the French say: "L'accessibilité n'est pas une option, c'est une nécessité fondamentale pour le web moderne."

Key Principles
Perceivable

Content must be presentable in ways users can perceive.

Operable

Interface must be operable by all users.

Understandable

Information must be understandable.
